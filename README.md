# 🧠 Learning-AI

A structured repository for understanding Artificial Intelligence concepts by implementing them **from scratch**, progressing from basic neural networks to Transformers.

---

## 📁 Project Structure

```
Learning-AI/
│
├── BPE/
│   └── bpe.ipynb                # Byte Pair Encoding tokenizer
│
├── Refresher/
│   ├── perceptron.ipynb        # Perceptron implementation from scratch
│   ├── scratch_mnist.ipynb     # ANN from scratch (no frameworks)
│   ├── mnist.ipynb             # MNIST experiments
│   ├── train.csv / test.csv    # MNIST dataset
│
├── Self Attention/
│   ├── attention.ipynb         # Self-attention implementation
│   ├── bpe.py                  # Reusable BPE tokenizer
│
├── Transformer/
│   └── transformer.ipynb       # Transformer implementation
│
└── README.md
```

---

## 📚 Modules

### Refresher

Covers fundamental neural network concepts:

* Perceptron implementation
* Artificial Neural Network (ANN) from scratch
* MNIST digit classification

---

### BPE (Byte Pair Encoding)

* Custom tokenizer implementation
* Converts raw text into tokenized form

---

### Self Attention

* Implementation of attention mechanism
* Includes Query, Key, Value computation
* Scaled dot-product attention

---

### Transformer

* Step-by-step implementation of Transformer architecture
* Built using concepts from previous modules

---

## ⚙️ Requirements

* Python 3.x
* NumPy
* PyTorch (optional, depending on notebook usage)
* Jupyter Notebook

---

## ▶️ Usage

```bash
git clone https://github.com/your-username/Learning-AI.git
cd Learning-AI
jupyter notebook
```
