# 🧠 Neural Network From Scratch (MNIST)

This project is a **from-scratch implementation of Artificial Neural Networks (ANN)** using Python — without using high-level libraries like TensorFlow or PyTorch.

It focuses on understanding the **core math and logic behind deep learning**.

---

## 🚀 Features

* 🔢 MNIST digit classification
* 🧠 Neural Network built from scratch
* ⚙️ Forward propagation & backpropagation
* 📉 Gradient descent optimization
* 🧮 Custom implementation of:

  * Sigmoid activation
  * Loss calculation
  * Weight updates
* 📊 Training & testing using CSV dataset

---


## ⚙️ How It Works

### 1. Data Input

* MNIST dataset loaded from CSV
* Pixel values normalized

---

### 2. Forward Propagation

* Input → Hidden layer → Output
* Activation applied (Sigmoid)

---

### 3. Loss Calculation

* Error between predicted and actual output

---

### 4. Backpropagation

* Compute gradients
* Update weights using gradient descent

---

### 5. Prediction

* Model predicts digit (0–9)
